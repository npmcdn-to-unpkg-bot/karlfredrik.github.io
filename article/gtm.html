Vad är GTM?

Publicerad 2016-04-5

Google Tag manager ( ofta förkortat till GTM ) är en teknologi från Google vilket möjliggör publicering av javascript/HTML-kod på en hemsida direkt ifrån Googles användargränssnitt. Det ger möjlighet till att själv kunna lägga in script på hemsidan utan att behöva blanda in en utvecklare.

Google erbjuder en rad förkonfigurerade typer av taggar, så som Analytics, Google Adwords och Linkedin. Du har även möjlighet att skapa dina egna taggar, där du kan skriva in egen HTML eller Javascript-kod.

Du har stora möjligheter att välja när den här koden ska köras, till exempel att den ska köras på varje sidvisning, varje gång ett formulär skickas in eller varje gång en användare klickar på en länk. Om du kan Javascript ( JS ) kan du även skicka in egna event genom Datalayer.

Datalayer

Datalayer är ett JS objekt med information som skickas till GTM. Den kan innehålla olika event och variabler. De kan sedan användas för att trigga en händelse i GTM.

Ett exempel på detta är hur jag har gjort för att koppla Analytics till Tag Manager. Jag binder den här koden på när någon klickar på min e-post.

dataLayer.push({
	'event': 'gaEvent',
	'eventCategory': "Mail",
	'eventAction': "click",
	'eventLabel': document.URL
});

Den här koden skickar in ett event med namnet gaEvent där kategorin är ”Mail”, action ”click” och label URL till sidan användaren var på när den triggade eventet.

I GTM skapar jag en Trigger ( Utlösare ) för eventet, se bild för detaljer.

Sedan skapar jag 3 användardefinierade variabler, för de jag skickade in via dataLayer.

Därefter skapar jag en tagg av typen : Produkt = Google Analytics.

När jag sedan går in under "Realtid" i Analytics kan jag se att informationen skickas genom GTM. Jag hoppas att detta kan hjälpa någon, och skulle ni ha någon feedback så går det bra att kontakta mig så kan jag uppdatera inlägget vid behov.
